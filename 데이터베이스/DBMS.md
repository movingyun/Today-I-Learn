## 데이터베이스의 필요성
### 데이터와 정보
 - 데이터 : 현실세계에서 단순히 관찰하거나 측정해 수집한 사실이나 값
 - 정보 : 의사결정에 유용하게 활용할 수 있도록 데이터를 처리한 결과물
 - 예시
	- 데이터 : A가 쿠팡에서 헤드폰을 검색하였다.
	- 정보 : Boss헤드폰과 샤오미 헤드폰을 반복적으로 검색하였다.
	- 지식 : 구매결정을 하기위한 고민을 하고있다.
	- 지혜 : A라는 사람에게 구매결정에 대한 상품 추천.
 - 차이점
	- 의미가 부여되었는가?
	- 의사결정에 사용할 수 있냐?

### 정보처리(Information processing)
 - **데이터**에서 **정보**를 추출하는 과정 또는 방법
 - 데이터 : 한빛 쇼핑몰 주문 내역
 - 정보 1) 제품별 총 판매액
		2) 분기별 총 판매액

### 정보 시스템과 데이터베이스
![image](https://user-images.githubusercontent.com/97611103/178094193-46a40ad5-ef1a-48aa-9716-14efd0be9a78.png)
#### 정보시스템(Information System)
 - 조직 운영에 필요한 데이터를 수집하여 저장해두었다가 필요할 때 유용한 정보를 만들어 주는 수단
 #### 데이터베이스
  - 정보시스템 안에서 데이터를 저장하고 있다가 필요할 때 제공하는 역할을 담당
  - **공동**으로 필요한 데이터를 **통합**하여 관리하는 데이터의 집합, 이를 통해 데이터가 **중복**되는 문제를 없애고 검색과 갱신의 **효율**을 높힌다.




## 데이터베이스의 정의와 특성
![image](https://user-images.githubusercontent.com/97611103/178093880-dffa8b8d-2036-4f63-88d3-0df2f4aa5313.png)
### 데이터베이스의 정의
 - 특정 조직의 여러 사용자가 **공유**하여 사용할 수 있도록 **통합**하여 **저장**한 **운영**데이터의 집합
 - 통합 : 최소한의 중복과 통제가능한 중복만 허용
 - 저장 : 컴퓨터가 접근할 수 있는 매체에 저장
 - 공유 : 여러 사용자가 함께 소유하고 이용할 수 있어야함.
 - 운영 : 조직의 기능을 수행하기 위해 지속적으로 유지

### 데이터베이스의 특성
 - **실시간 접근** : 사용자의 데이터 요구에 실시간으로 응답
 - **계속 변화** : 데이터의 계속적인 CRUD를 통해 현재의 정확한 데이터를 유지
 - 내용 기반 참조 : 데이터가 저장된 주소나 위치가 아닌 내용으로 참조 ex)연봉>5000 & 직급 > 과장
 - 동시 공유 : 서로 다른 데이터의 동시 사용뿐만 아니라 같은 데이터의 동시 사용도 지원
