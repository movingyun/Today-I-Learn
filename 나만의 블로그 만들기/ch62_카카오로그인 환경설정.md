## ch62_카카오 로그인 환경설정
### 2022.07.13

### ✔ 카카오 로그인 구현 사전 설정
 - https://developers.kakao.com/ 접속하기
 - 내 애플리케이션 추가 -> REST API키 가져오기
 - 플랫폼(Web) 설정하기 -> http://localhost:8080 입력
 - 좌측 카카오 로그인탭 들어간 후 활성화하기 -> Redirect URI 입력(http://localhost:8080/auth/kakao/callback)
 - 동의항목 설정(프로필 사진(필수동의), 이메일(선택동의))
